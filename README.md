# ПромОриентир — платформа промышленного туризма

**ПромОриентир** — веб-платформа для поиска, сравнения и бронирования промышленных экскурсий по России. Позволяет предприятиям публиковать свои экскурсионные программы, а туристам — находить и бронировать их.

---

## Функциональные возможности

### Для туристов
- **Каталог предприятий** — поиск и фильтрация по региону, типу производства, возрасту, длительности экскурсии
- **Карта России** — интерактивная карта с кластеризованными метками предприятий и границами регионов
- **Карточка предприятия** — описание, фотогалерея, видео, панорама 360°, сертификаты, сувениры, профессии
- **Экскурсии** — просмотр доступных экскурсий с указанием цены, длительности, уровня интерактива
- **Бронирование** — отправка заявки на экскурсию
- **AI-ассистент** — умный помощник для подбора экскурсий на основе предпочтений
- **Шкала сравнения** — сравнение предприятий по различным параметрам
- **Промышленная миля** — формирование маршрута из нескольких предприятий

### Для предприятий (личный кабинет)
- **Паспорт предприятия** — редактирование информации, фотографий, видео, сертификатов
- **Управление экскурсиями** — создание, редактирование, публикация экскурсий
- **Аналитика** — просмотр статистики по заявкам и просмотрам

### Для администратора
- **Панель управления** — навигация через боковое меню:
  - 📊 Обзор — статистика по предприятиям, заявкам, пользователям
  - 🏭 Предприятия — создание, редактирование (с картой, тегами), модерация, удаление
  - 🎫 Экскурсии — полный CRUD со всеми полями
  - 📋 Заявки — управление статусами бронирований
  - 👥 Пользователи — управление ролями и активностью
  - 📍 Места — управление местами размещения, питания и досуга
  - 🗺️ Регионы — управление информацией о регионах
  - ⚙️ Настройки — AI-ассистент, системный промпт, параметры генерации, VK-уведомления
- **VK-уведомления** — оповещения о новых заявках, регистрациях, изменениях статуса предприятий
- **Теги предприятий** — множественный выбор из предопределённого списка
- **Редактирование на карте** — перетаскиваемый маркер для установки координат

## Технологический стек

### Frontend
- **React 18** + **Vite** + **Tailwind CSS**
- **React Router** для клиентской маршрутизации
- **Яндекс.Карты API** — интерактивная карта с полигонами регионов и кластеризатором
- **OpenAI SDK** — AI-ассистент

### Backend
- **Node.js** + **Express.js**
- **SQLite3** — база данных
- **JWT** — аутентификация
- **bcrypt** — хеширование паролей
- **express-rate-limit** — ограничение запросов

### Инфраструктура
- **Docker** + **Docker Compose** — контейнеризация
- Многозвеночная сборка (multi-stage build) — фронтенд собирается внутри контейнера

## Быстрый запуск

### Запуск в Docker (рекомендуется)

```bash
# Клонировать репозиторий
git clone <repo-url>
cd 19

# Собрать и запустить
docker compose up -d

# Приложение доступно по адресу:
# http://localhost:17626
```

### Локальный запуск (для разработки)

```bash
# Терминал 1: запуск сервера
cd server
npm install
npm run dev

# Терминал 2: запуск клиента
cd client
npm install
npm run dev

# Клиент: http://localhost:5173
# Сервер: http://localhost:3000
```

## Разработка

### Создание новой сущности

1. Создать миграцию в `server/src/db/init-db.js`
2. Добавить SQL в `server/src/db/schema.sql`
3. Создать маршруты в `server/src/routes/`
4. Добавить методы API в `client/src/api/index.js`
5. Создать компоненты в `client/src/components/`

### VK-уведомления

Настраиваются в панели администратора (⚙️ Настройки → VK-уведомления):
- Указать токен сообщества VK
- Указать peer_id администратора
- Тестировать через кнопку "📨 Тест"

Уведомления автоматически отправляются при:
- Новой заявке на экскурсию
- Регистрации нового пользователя
- Изменении статуса предприятия
